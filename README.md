Travel Bucket List – CRUD Application
INFR3120 – Assignment 3

Overview
This application is a Travel Bucket List CRUD system. Users can create, view, update, and delete travel adventures.
The project includes full cloud deployment, Bootstrap styling, version control, and secure environment variable handling.
